# Hydra::Shibboleth

A Rails engine to add Shibboleth auth with LDAP/Active Directory groups.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hydra-shibboleth'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hydra-shibboleth
